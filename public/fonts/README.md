# תיקיית פונטים

## הוספת פונט "דנה יד"

1. הורד את פונט "דנה יד" מ-https://freefonts.co.il/
2. פתח את קובץ ה-ZIP שהורדת
3. העתק את קבצי הפונט (TTF, OTF, או WOFF) לתיקייה זו
4. שנה את שם הקובץ ל-`DanaYad.ttf` (או `.otf` / `.woff` בהתאם לסוג הקובץ)

### שמות קבצים נתמכים:
- `DanaYad.ttf`
- `DanaYad.otf`
- `DanaYad.woff`
- `DanaYad.woff2`

הפונט יטען אוטומטית לאחר הוספת הקבצים.
